# Karl · AI 产品经理作品集

线上地址：<https://hzj.wxditor.chat>

作品集包含 WXEditor、AIGC 先锋学会和 ARK AI 家居导购三个案例，重点展示产品界面、用户链路、系统架构和指标设计。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

仓库已包含 ARK 与 WXEditor 的静态嵌入产物。仅构建作品集站点时运行：

```bash
npm run build:site
```

`npm run build` 用于本机同步 ARK、WXEditor 原项目后再构建完整站点。
